Voici la liste des potentielles fonctions des substances analysées dans le cadre du suivi des masses d'eau souterraines : Acaricide : action contre les acariens Biocide : famille regroupant l'ensemble des substances ayant une action pour détruire, repousser ou rendre inoffensifs des organismes nuisibles. Entre autres, les pesticides, les antimicrobiens (antibiotiques par exemple) et les désinfectants appartiennent à cette famille Fongicide : action sur les champignons, la moisissure (contre le mildiou par exemple) Graminicide : action sur les graminées, appartient à la famille des herbicides Herbicide : action contre les végétaux comme les mauvaises herbes Insecticide : action contre les insectes, leurs larves ou leurs oeufs Molluscicide : action contre les mollusques tels que les escargots ou les limaces Nématicide : action contre les nématodes tels que les vers de terre Régulateur de croissance : action pour ralentir la croissance d'une plante Répulsif : permet de repousser les animaux (insectes ou mammifères par exemple) Rodenticide : action contre les rongeurs A noter que : - une substance unique peut avoir plusieurs fonctions, - certaines substances métabolites ainsi que certaines substances proches groupées n'ont pas de fonction stricto-sensus.
